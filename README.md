# EasyCpp

This is the VS Code snippets extension for easy creating hpp, cpp (with or witout implemented pimpl idiom). Also this snippets could be used to crate basic CMakeLists files.

## Usage

cpp  
cpp-pimpl  
hpp  
hpp-pimpl  
cmake-project  
cmake-tests
